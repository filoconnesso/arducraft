### SETUP

Install **pyserial**, **tabulate** and **asyncio** modules

On Windows :

```sh
pip install pyserial
```
```sh
set TABULATE_INSTALL=lib-only
pip install tabulate
```
```sh
pip install asyncio
```

On MacOS : 

```sh
pip install pyserial
```
```sh
pip install tabulate
```
```sh
pip install asyncio
```

Launch the daemon

```sh
py ./arducraft.py
```
- Upload the sample sketch
- Select the device COM
- Enjoy!
